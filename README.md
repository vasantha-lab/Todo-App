# Todo App

A simple Todo Application built using React.js. This project helps users organize their daily tasks by allowing them to add and delete todos dynamically. It demonstrates the use of React Functional Components, Hooks (`useState`), event handling, and list rendering.

## Features
- Add new tasks
- Delete existing tasks
- Real-time UI updates
- Responsive and user-friendly interface

## Technologies Used
- React.js
- JavaScript (ES6+)
- HTML5
- CSS3
- Bootstrap

## Learning Outcomes
- React Functional Components
- useState Hook
- Event Handling
- State Management
- List Rendering with map()
